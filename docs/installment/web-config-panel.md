# OpenClaw 网页配置面板设计方案

## 文档概述

本文档描述 OpenClaw 网页配置面板的产品设计方案，用于提升用户体验，特别是针对首次使用和小白用户场景。

**设计理念：** 渐进增强 - 网页配置为主（简单场景），Samba 挂载为辅（高级场景）

**状态：** 📋 方案设计阶段，尚未实施

---

## 用户场景分析

### 首次使用流程

```
1. 用户给开发板上电
   ↓
2. 开发板启动 AP 模式（热点，如 OpenClaw-Setup）
   ↓
3. 用户手机/电脑连接热点
   ↓
4. 打开网页（自动弹出或提示访问 http://192.168.4.1）
   ↓
5. WiFi 配置向导：
   - 搜索附近 WiFi
   - 选择网络
   - 输入密码
   - 连接
   ↓
6. 连接成功后显示：
   - 开发板 IP 地址
   - 下一步操作指引
   ↓
7. 用户切换到家庭 WiFi，继续配置
```

### 日常使用场景

| 场景类型      | 推荐方式 | 说明               |
| ------------- | -------- | ------------------ |
| 快速查看状态  | 📱 网页  | 随时随地，手机也可 |
| 简单配置调整  | 📱 网页  | 模型、API Key 等   |
| 配置文件编辑  | 📱 网页  | 带验证和语法高亮   |
| 查看历史会话  | 📱 网页  | 浏览和搜索         |
| 系统诊断      | 📱 网页  | Doctor 功能        |
| 批量文件编辑  | 💻 Samba | 使用本地工具       |
| 专业 IDE 操作 | 💻 Samba | VS Code 等         |
| 备份/恢复     | 💻 Samba | 大量数据操作       |

### API 接口设计

```typescript
// 现有的 Gateway Server 扩展
interface GatewayAPI {
  // WiFi 配置
  GET  /api/wifi/status           // 获取 WiFi 状态
  POST /api/wifi/scan             // 扫描附近 WiFi
  POST /api/wifi/connect          // 连接 WiFi

  // 配置管理
  GET  /api/config                // 获取配置
  PUT  /api/config                // 更新配置
  GET  /api/config/validate       // 验证配置
  GET  /api/config/backups        // 获取备份列表
  POST /api/config/restore        // 恢复备份

  // 文件操作
  GET  /api/files/list            // 列出文件
  GET  /api/files/read            // 读取文件
  PUT  /api/files/write           // 写入文件

  // 会话管理
  GET  /api/sessions              // 列出会话
  GET  /api/sessions/:id          // 获取会话详情
  DELETE /api/sessions/:id        // 删除会话
  GET  /api/sessions/:id/export   // 导出会话

  // 系统监控
  GET  /api/system/stats          // 系统状态
  GET  /api/system/processes      // 进程列表
  GET  /api/system/logs           // 系统日志

  // 诊断
  POST /api/diagnostics/run       // 运行诊断
  GET  /api/diagnostics/results   // 诊断结果
}
```

## 与 Samba 的协同

### 功能定位

| 功能     | 网页      | Samba     |
| -------- | --------- | --------- |
| 快速查看 | ✅ 推荐   | ⚠️ 可用   |
| 简单编辑 | ✅ 推荐   | ⚠️ 可用   |
| 批量操作 | ❌ 不适合 | ✅ 推荐   |
| 专业工具 | ❌ 不支持 | ✅ 推荐   |
| 移动访问 | ✅ 支持   | ⚠️ 有限   |
| 实时预览 | ✅ 支持   | ❌ 不支持 |
